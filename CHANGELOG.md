# Changelog

All notable changes to this project will be documented in this file.

This project follows a simple changelog format and intends to use semantic versioning once published.

## 0.1.0 - 2026-06-01

### Added

- Operational transport code formatting, parsing, and validation.
- Invoice number normalization, parsing, and validation.
- Calendar-date matching for ride and invoice service dates.
- Ride grouping by normalized invoice number.
- Basic ride/invoice link validation helper.
- TypeScript build, Vitest tests, README, MIT license, and contribution notes.
