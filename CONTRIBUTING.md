# Contributing

Thank you for considering a contribution to `small-transport-ops-toolkit`.

## Scope

This project is a small public TypeScript utility library for generic transport operations workflows. Contributions should stay generic and should not include private operator workflows, private datasets, real passenger names, driver names, vehicle registrations, customer names, school names, phone numbers, emails, internal routes, or copied private code.

## Development

Install dependencies:

```bash
npm install
```

Run tests:

```bash
npm test
```

Run type checking:

```bash
npm run typecheck
```

Build the package:

```bash
npm run build
```

## Pull Request Guidelines

- Keep changes small and reviewable.
- Add or update tests for behavior changes.
- Prefer clear error messages over silent failures.
- Document public API changes in `README.md`.
- Update `CHANGELOG.md` for user-visible changes.

## Privacy and Safety

Do not submit private transport company data or private business logic. Use generic examples only.
